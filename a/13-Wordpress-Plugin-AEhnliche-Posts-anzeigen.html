Wordpress Plugin: Ähnliche Posts anzeigen

In vielen Blogs sieht man unter den Beiträgen eine Liste von Postings die einen ähnlichen Inhalt haben. Das hat zweierlei Vorteile: Zuerst ist es gut für einen Besucher der kein regelmäßiger Leser ist und über eine Suchmaschine oder einen Link im Blog landet. Er findet so problemlos weitere Beiträge die ihn interessieren könnten.
Zum anderen bewerten Suchmaschinen seit einiger Zeit auch die interne Struktur einer Seite, werden thematisch passende Dinge gut intern verlinkt kann das helfen da die Suchmaschine hier gut erreichbare Inhalte sieht.


Um automatisch Links zu verwandten Artikeln zu erstellen gibt es für Wordpress ein einfaches Plugin namens "Wordpress Related Entries". Die Installation geht völlig problemlos von statten: Runterladen, entpacken, auf den Webspace hochladen, Plugin im Admin aktivieren, das wars.
Nach der Installation findet sich unter Plugins ein neuer Tab "Related Posts Options", in diesem muss man nach der ersten Installation einmal per Klick die Datenbank für das Plugin einrichten. Der Link dazu findet sich am Ende der Seite.
War dieser Schritt erfolgreich kann man einstellen wie viele Related Posts jeweils angezeigt werden sollen dort kann man einstellen wie viele verwandte Beiträge angezeigt und wie diese formatiert werden sollen. Als letzten Schritt muss noch <?php related_posts() ?> und ein geeigneter Hinweis in die Wordpress-Loop (in diesem Fall meist post.php) eingefügt werden. Alternativ macht sich das Plugin auch in der Sidebar recht gut.
