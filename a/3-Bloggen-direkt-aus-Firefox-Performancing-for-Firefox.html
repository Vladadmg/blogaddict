Bloggen direkt aus Firefox: Performancing for Firefox

Performancing.com hat sich innerhalb kürzester Zeit einen Namen als Anlaufstelle für professionelle Blogger gemacht. Einer der Gründe dafür ist definitiv Performancing for Firefox. PFF wird als Extension in Firefox installiert und dient dann als Blog-Editor für beliebig viele Blogs direkt aus dem Browser.


Um PFF aufzurufen gibt es mehrere Möglichkeiten. Der schnellst Weg führt über die F8-Taste, dann nimmt PFF ungefähr die halbe Höhe des Browser-Fensters ein, das ist ideal wenn man über die gerade aufgerufene Seite schreiben will. Alternativ kann man PFF auch in einem Tab oder als neues Fenster öffnen.

Die Einrichtung von Blogs geht problemlos und schnell von der Hand: Einfach in der Blogliste auf "Add" klicken und die URL des Blogs angeben. Die verwendete API wird fast immer automatisch erkannt und es müssen nur noch die Zugangsdaten ergänzt werden.

Für das Schreiben von Beiträgen steht ein WYSIWYG-Editor mit den üblichen Buttons für Schriftformatierung und -Ausrichtung zur Verfügung. Bilder können per API oder FTP auf das Blog oder auf Flickr hochgeladen werden. Schreiben kann man entweder in einem Rich-Editor der auch Formatierungen anzeigt oder im HTML-Quellcode. Zusätzlich kann eine Live-Preview aufgerufen werden, deren Anzeige unterscheidet sich aber nicht wesentlich von der des Rich-Editors. Den Quellcode braucht man leider recht oft, denn Bilder von Text umfließen zu lassen oder auch nur einen Link im neuen Fenster zu öffnen übersteigt derzeit noch die Fähigkeiten von PFF.

Optional können Postings als Entwurf erzeugt, Technorati-Tags hinzugefügt und der Beitrag direkt bei del.icio.us gebookmarkt werden. Ein Formularfeld zum Senden von Trackbacks ist vorgesehen, zumindestens mit Wordpress wurden diese bei mir aber bisher nicht abgeschickt. Über das neue Posting kann die Blog-Welt per Pingo-O-Matic, PingoAt oder einen anderen Ping-Dienst informiert werden.

Performancing bietet derzeit unter dem Namen Performancing Metrics auch noch Webstatistiken an, auch diese können für die eigenen Seiten in sehr begrenztem Maß direkt aus PFF eingesehen werden. Wie lange das noch möglich ist, ist fraglich denn derzeit plant Performancing einen Verkauf des Dienstes.

Weiterhin können die eigenen del.icio.us-Bookmarks verwaltet und Seiteninformationen über Technorati eingesehen werden. Über Plugins ist PFF zudem erweiterbar, die Auswahl ist bisher aber knapp.
