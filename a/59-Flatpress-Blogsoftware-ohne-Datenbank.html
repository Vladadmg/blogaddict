Flatpress: Blogsoftware ohne Datenbank


Auch wenn der Name es vermuten lässt, in der Funktion kann Flatpress dem großen Bruder Wordpress nicht das Wasser reichen, so fehlt zum Beispiel bisher das komplette Plugin-System. Eine Verwandschaft gibt es aber dennoch: Flatpress basiert auf Wordpress-Code, hat sich aber zum Hauptziel gesetzt eine Blogsoftware ohne Datenbank zu werden. Somit benötigt Flatpress kein installiertes MySQL sondern speichert alle Beiträge in sog. Flatfiles, also einfachen Textdateien.


Neben diesem Hauptfeature gibt es noch einige kleinere Unterschiede. Der wohl größte ist, dass Flatpress die Smarty-Engine für Templates verwendet und somit - im Gegensatz zu Wordpress - eine tatsächliche
Template-Engine hat.

Dadurch, dass Flatpress kein MySQL benötigt kann es als Einstiegssystem durchaus taugen. Der geringere Funktionsumfang kommt dem Bloganfänger vermutlich ebenso entgegen wie die geringeren technischen Anforderungen. Monetär sollte ein MySQL heutzutage kein Problem mehr sein, fast jeder Hoster hat für wenig Geld
entsprechende Angebote, aber technisch kann man sicherlich davor zurückschrecken. Außerdem - das ist auch eine der Motivationen des Autors von FlatPress - mag es beruhigen, dass alle Postings im Klartext auf dem Server liegen. Sollte es einmal Probleme mit der Software oder dem Server geben können die Daten so notfalls sogar von Hand in ein neues Blog übertragen werden da sie näherungsweise lesbar vorliegen.

Weitere Informationen zu Flatpress finden sich auf der Entwicklungsseiten FlatDev
