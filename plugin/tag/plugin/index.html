Wordpress Plugin: Breadcrumb Navigation


Navigation mit Brotkrümeln? Kling komisch, bezeichnet aber schlichtweg eine Funktion die die Position der aktuellen Webseite innerhalb der Gesamtstruktur anzeigt. Heraus kommt dann bspw. soetwas:

Blogaddict > Februar 2007 > Wordpress Plugin: Breadcrumb Navigation

Das ganze erfüllt zweierlei Zweck:

  1. erleichtert es Besuchern die Orientierung auf der Seite. Gerade wenn die Struktur nicht zeitlich sondern über Kategorien abgebildet ist (also anders als in meinem Beispiel oben), ist es durch die Breadcrumb mit einem Klick möglich thematisch ähnliche Beiträge in der Kategorie zu finden
  2. die interne Verlinkung der Seite wird verbessert und das Blog dadurch eventuell besser in Google und anderen Suchmaschinen gefunden

In einem Wordpress-Blog lässt sich eine Breadcrumb Navigation ganz einfach über das gleichnamige Plugin erzielen. Nach der Installation sind allerdings einige Anpassungen an den Blogtemplates nötig um die Brotkrumen einzufügen.

Alternative zu Captcha-Bildern


Lange Zeit haben uns die Captcha-Bildchen unter fast jedem Kommentarfeld verfolgt. Oft war auch für das menschliche Auge kaum erkennbar, welcher Satz oder welche Zeichen sich hinter der Grafik verstecken sollten.

Seit einigen Wochen setzen immer mehr Blogs auf die Spambekämpfung über Mathe-Aufgaben, ein paar Worte dazu gab es auch hier schon.

Bei website-spy gab es vor geraumer Zeit einen schönen Beitrag wie man den Großteil des Spamproblems lösen kann, ohne es dem "Otto Normal-Kommentierer" schwerer zu machen: erstens lohnt es alle Formularfelder umzubenennen. Dadurch scheitern viele Spambots schon beim ausfüllen des Kommentarfeldes beziehungsweise finden es überhaupt nicht mehr. Außerdem wird nach einem festgelegten Schlüssel ein täglich wechselndes Formularfeld eingefügt.

Die Idee ist herrlich einfach und effizient: Selbstverständlich lässt sich auch dieser Schutz umgehen, erfordert aber zumeist eine manuelle Anpassung für jedes Blog und führt dadurch einen der Grundgedanken von Blogspamming - nämlich die Automatisierung - ad absurdum. Einziger Nachteil: Zumindestens rudimentäre Programmierkenntnisse können nicht schaden um die Lösung überhaupt umsetzen zu können.

Wordpress Plugin für ähnliche Posts verbessert


Das hier bereits vorgestelle Wordpress-Plugin für ähnliche Beiträge hat einen Haken: Um die Beiträge zu vergleichen wird nur der Inhalt der Url genutzt und das ist bei umgeschriebenen Urls (mod_rewrite) nicht immer sehr effektiv. Das hat sich jedenfalls Peter Boyer gedacht und daher eine verbesserte Version programmiert die eine Ähnlichkeit aufgrund eines gewichteten Vergleichs von URL, Titel und Inhalt errechnet.
Das Improved Wordpress related posts plugin kann man auf der Webseite von Peter Bowyer herunterladen.

Intelligente 404-Seite für Wordpress

Mit wachsender Größe eines Blogs lassen sich 404-Fehler nicht vermeiden. Es wird vorkommen, dass jemand falsche Links setzt, dass User auf kaputte Links z.B. in Emails klicken oder das schlichtweg mal ein Posting gelöscht oder wegen Tippfehlern die URL geändert wird.
In jedem dieser Fälle bekommt der geneigte Besucher normalerweise eine recht simple 404-Seite vorgesetzt die ihm mitteilt, dass er nichts gefunden hat. Ideal wäre es natürlich, wenn stattdessen eine hilfreiche Seite angezeigt würde.

Wordpress Plugin: Automatisch zu YiGG posten

Im englischsprachigen Raum ist digg.com der absolute Garant für jeden Server ruinierenden Traffic. Das deutsche Pendant YiGG kann vom Traffic noch nicht mithalten, ist aber trotzdem eines der besten Social Bookmarking Portale in Deutschland.
Mit Wordpress funktioniert das Einstellen von Beiträgen bei YiGG besonders einfach: Es gibt ein Plugin das jeden neuen Beitrag oder auch ausgewählte Beiträge per Mausklick über eine API an YiGG meldet. Man muss lediglich bei der Post-Erstellung die gewünschte Kategorie und einige passende Tags angeben.
Auch wenn die Bandbreite der Themen bei YiGG sehr breit ist, lohnt sich die Einstellung jedes (!) Artikels im Moment bei Technik-lastigen Themen sicherlich am meisten da die Hauptnutzerschaft aus diesem Bereich zu kommen scheint.

Wordpress Plugin: Ähnliche Posts anzeigen

In vielen Blogs sieht man unter den Beiträgen eine Liste von Postings die einen ähnlichen Inhalt haben. Das hat zweierlei Vorteile: Zuerst ist es gut für einen Besucher der kein regelmäßiger Leser ist und über eine Suchmaschine oder einen Link im Blog landet. Er findet so problemlos weitere Beiträge die ihn interessieren könnten.
Zum anderen bewerten Suchmaschinen seit einiger Zeit auch die interne Struktur einer Seite, werden thematisch passende Dinge gut intern verlinkt kann das helfen da die Suchmaschine hier gut erreichbare Inhalte sieht.
