Wordpress Plugin: Breadcrumb Navigation


Navigation mit Brotkrümeln? Kling komisch, bezeichnet aber schlichtweg eine Funktion die die Position der aktuellen Webseite innerhalb der Gesamtstruktur anzeigt. Heraus kommt dann bspw. soetwas:

Blogaddict > Februar 2007 > Wordpress Plugin: Breadcrumb Navigation

Das ganze erfüllt zweierlei Zweck:

  1. erleichtert es Besuchern die Orientierung auf der Seite. Gerade wenn die Struktur nicht zeitlich sondern über Kategorien abgebildet ist (also anders als in meinem Beispiel oben), ist es durch die Breadcrumb mit einem Klick möglich thematisch ähnliche Beiträge in der Kategorie zu finden
  2. die interne Verlinkung der Seite wird verbessert und das Blog dadurch eventuell besser in Google und anderen Suchmaschinen gefunden

In einem Wordpress-Blog lässt sich eine Breadcrumb Navigation ganz einfach über das gleichnamige Plugin erzielen. Nach der Installation sind allerdings einige Anpassungen an den Blogtemplates nötig um die Brotkrumen einzufügen.
